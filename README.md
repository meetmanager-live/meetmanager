# MEET.MANAGER — Tampilan Lomba Renang

Situs statis (1 file HTML) untuk menampilkan acara lomba renang yang sedang
berlangsung beserta starting list-nya, plus panel admin untuk mengelola
data lewat upload CSV. Siap dihosting gratis di **GitHub Pages**.

📖 **Panduan lengkap (cara hosting, menu viewer/admin, format CSV) ada di
[`TUTORIAL.md`](./TUTORIAL.md).**

## Isi repo ini

```
index.html      → aplikasi utama (viewer + panel admin)
img/bg.jpg       → gambar background (ganti dengan fotomu sendiri)
TUTORIAL.md      → panduan lengkap pemakaian
```

## Cara cepat hosting ke GitHub Pages

1. Upload semua file di repo ini (`index.html`, folder `img/`) ke repository
   GitHub kamu.
2. Buka **Settings → Pages** di repo tersebut.
3. Source: pilih branch `main`, folder `/ (root)` → **Save**.
4. Situs akan online di `https://namauser-kamu.github.io/nama-repo/`.

Detail lengkap + cara ganti password admin, ganti background, dan format
upload CSV: lihat [`TUTORIAL.md`](./TUTORIAL.md).
